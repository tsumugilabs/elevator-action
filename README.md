# Elevator Action — ブラウザ版

2Dアクションの名作『エレベーターアクション』にインスパイアされた、
HTML5 Canvas + 素のJavaScriptによるブラウザゲームです。ビルド不要で、
`index.html` を開くだけで遊べます。

## 遊び方

スパイとなってビル内に潜入し、**赤いドアの機密書類をすべて回収**して、
地下の脱出ゲート（EXIT）まで到達すればミッション成功です。エレベーターや
足場を渡り、敵エージェントの銃撃をかわしながら攻略します。

### 操作

| キー | 動作 |
| --- | --- |
| `←` `→` | 左右移動 |
| `↑` `↓` | エレベーター乗降 / しゃがみ |
| `Z` | ジャンプ |
| `X` / `Space` | ショット |
| `Enter` | スタート / リトライ |

### ルール

- **赤いドア** … 機密書類。触れると回収（+500点）。
- **その他のドア** … 一部から敵エージェントが出現。
- **エレベーター** … 乗って上下フロアを移動。上昇するエレベーターと天井に
  挟まれるとミス。
- 敵の弾・接触でミス。ライフが尽きるとゲームオーバー。
- 全書類を回収して EXIT に到達でクリア（残機ボーナスあり）。

## 起動方法

追加の依存やビルドはありません。

```bash
# 方法1: ファイルを直接開く
open index.html            # macOS
xdg-open index.html        # Linux

# 方法2: ローカルサーバー経由（推奨）
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

## 構成

```
elevator-action/
├── index.html         # エントリーポイント（HUD・オーバーレイ）
├── css/style.css      # スタイル
└── js/
    ├── input.js       # キーボード入力（押下・エッジ検出）
    ├── entities.js    # プレイヤー / 敵 / 弾 と AABB 物理ヘルパー
    ├── level.js       # ビル構造・エレベーター・ドア・描画
    └── game.js        # メインループ / 状態管理 / 当たり判定 / カメラ
```
