# Elevator Action — ブラウザ版

2Dアクションの名作『エレベーターアクション』にインスパイアされた、
HTML5 Canvas + 素のJavaScriptによるブラウザゲームです。ビルド不要で、
`index.html` を開くだけで遊べます。

## 遊び方

スパイとなってビル内に潜入し、**赤いドアの機密書類をすべて回収**して、
地下の脱出ゲート（EXIT）まで到達すればミッション成功です。エレベーターや
足場を渡り、敵エージェントの銃撃をかわしながら攻略します。

### 操作

| キー | 動作 |
| --- | --- |
| `←` `→` | 左右移動 |
| `↑` `↓` | エレベーター乗降 / しゃがみ |
| `Z` | ジャンプ |
| `X` / `Space` | ショット |
| `Enter` | スタート / リトライ |
| `M` | サウンド ON/OFF（画面右上の🔊ボタンでも切替可） |

**サウンド**：BGMと効果音（ショット・書類回収・敵撃破・警告・ミス等）を
Web Audio APIでその場合成しています（音声ファイル不要）。ブラウザの仕様上、
STARTを押した瞬間から再生されます。

**スマホ・タブレット対応**：タッチ端末では画面下部に十字キーとアクションボタン
（撃つ / 跳ぶ）が自動表示され、タップで操作できます。

### ステージ（全6面・難易度上昇）

進むほど敵が手強くなります。各面クリアで次の面へ（スコア・残機は引き継ぎ）。

| 面 | 難易度 | 書類 | 敵最大同時 | 追従 | 敵弾 | 敵の上下移動 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | TUTORIAL | 3 | 1 | なし | なし | なし |
| 2 | BEGINNER | 4 | 2 | なし | なし | なし |
| 3 | EASY | 5 | 3 | あり | なし | なし |
| 4 | NORMAL | 6 | 3 | あり | 最下層の1体のみ | あり |
| 5 | HARD | 7 | 4 | あり | 同時2体まで | あり |
| 6 | EXPERT | 8 | 5 | あり | 全員 | あり（制限なし） |

### ルール

- **赤いドア** … 機密書類。触れると回収（+500点）。
- **その他のドア** … 一部から敵エージェントが出現。
- **エレベーター** … 乗って上下フロアを移動。上昇するエレベーターと天井に
  挟まれるとミス。**左シャフトは低速・右シャフトは高速**。
- **階段** … 上階／下階の移動手段。階段の上で `↓`（下り）、下で `↑`（上り）。
  最下層（地下）へ降りる階段もあり。**最終ステージでは敵も階段を使って追ってくる**。
- **スナイパー（4面の特殊敵）** … 一定時間ごとに、**プレイヤーが今いた場所**へ狙撃。
  サイトが徐々に絞られ、撃つ直前に**赤く点滅**して発砲。動いていれば当たらない
  （5面から手に入る OKB 13 の布石）。
- **強化アイテム** … 敵を倒すとランダムでドロップ。触れて取得。
  - 🔫 **マシンガン**（金の箱）… 取得数でレベルアップ。Lv1 前方連射 → Lv2 前後同時 →
    以降 Lv8 まで**8方向発射** → Lv9 で1方向に戻るが**敵をホーミング（必中）**。
  - 🛡 **防弾チョッキ**（緑の箱）… 被弾を肩代わり。取得で+3、**最大10**まで加算。
- **ライフ制** … ライフ2からスタート。被弾で1つ減り、**3回目の被弾でゲームオーバー**。
- **コンティニュー** … ゲームオーバー時に「CONTINUE」で**現在のステージから再開**
  （スコア引継ぎ）。再開前にランダムの煽り文句を6秒表示（3秒間は英語、その後
  日本語訳がフェイドイン）。**4回目以降のコンティニューでは**「プライドをゴミ箱に
  捨てますか？」の選択が出て、YES でフル装備（防弾チョッキ10・マシンガン8方向・
  地雷・OKB 13）で再開できる。
- **装備の持ち越し** … 特殊武器・防弾チョッキは**ステージをまたいで持ち越し**。
  クリア毎の選択は**加算**（既存の装備に上乗せ）。
- **装備クリア毎の選択** … 次ステージ開始前に装備を1つ追加。
  - 💣 **地雷**（特殊武器）… **専用ボタン（C / 画面の「地雷」ボタン）**で足元に設置。
    銃とは別に撃てる。設置後まもなく起爆状態になり、触れた敵と周囲の敵を爆風で撃破
    （自分も爆風に注意）。
- 🎯 **OKB 13**（5面以降の選択画面で入手可・非ドロップ・弾数無限）… ゴルゴ13
  パロディのスナイパー。**画面の敵をタップ/クリックすると即キル**。選択時には
  使い方の説明カットを表示。発動中はゲーム時間が止まり、約3秒かけて敵にズーム→
  ロックオン→劇画調スナイパーの「有罪（ギルティ）」カット（1秒）→ヘッドショット
  →余韻（約0.75秒のスロー引き）というドラマチックな演出が入る。

### デバッグモード

ステージや特殊武器を指定して開始できます（検証用）。

- PC: バッククォート **`` ` ``** キーでデバッグメニューを開く
- スマホ: URL末尾に **`#debug`** を付けて開くと、HUDに「DBG」ボタンが出現
- メニューで STAGE（1〜6）と WEAPON（なし/🔫/🛡/🎯OKB 13）を選んで GO
- 全書類を回収して EXIT に到達でクリア（残機ボーナスあり）。

## 起動方法

追加の依存やビルドはありません。

```bash
# 方法1: ファイルを直接開く
open index.html            # macOS
xdg-open index.html        # Linux

# 方法2: ローカルサーバー経由（推奨）
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

## 構成

```
elevator-action/
├── index.html         # エントリーポイント（HUD・オーバーレイ）
├── css/style.css      # スタイル
└── js/
    ├── input.js       # キーボード入力（押下・エッジ検出）
    ├── entities.js    # プレイヤー / 敵 / 弾 と AABB 物理ヘルパー
    ├── level.js       # ビル構造・エレベーター・ドア・描画
    └── game.js        # メインループ / 状態管理 / 当たり判定 / カメラ
```
